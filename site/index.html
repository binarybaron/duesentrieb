Duesentrieb

A fork of the Pi agent harness — an interactive, self-extensible coding agent for your terminal.

What the fork adds

  • Permission modes — manual, read-only, and auto (read-only) modes with an LLM permission classifier.
  • Open questions — a background model collects non-urgent questions during a session; review and answer them anytime with /open-questions.
  • Zellij integration — session forks open in Zellij panes.
  • Subagents — subagents inherit the parent session's permissions.
  • Skill discovery — picks up Claude project skills automatically.
  • No telemetry — telemetry is disabled entirely.

Install

Requires Node.js ≥ 22.19 and git.

git clone https://github.com/binarybaron/duesentrieb
cd duesentrieb
npm install --ignore-scripts
./scripts/install-fork.sh

This builds all packages and installs the pi command to ~/.local/bin (or over an existing pi install). Then just run:

pi
Prefer the unmodified upstream? Install it with npm install -g @earendil-works/pi-coding-agent or visit pi.dev.
