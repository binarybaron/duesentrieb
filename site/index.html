dusentrieb-cli

Duesentrieb is a self-extensible coding agent that lives in your terminal — where you already were anyway.

Asks Before It Wrecks

> /permissions
Permission mode: auto (read only)

Manual, read-only, and auto modes with an LLM classifier deciding what's safe. Your rm -rf anxiety, outsourced.

Holds Its Questions

> /open-questions
2 open questions collected

A background model collects non-urgent questions while you work. Answer them when you feel like it, not when a modal says so.

Forks Like It Means It

> /fork
Opened fork in Zellij pane

Session forks open in Zellij panes. Subagents inherit your permissions. Telemetry was removed entirely, because obviously.

Go get it

Requires Node.js ≥ 22.19 and git:

> git clone https://github.com/binarybaron/duesentrieb
> cd duesentrieb
> npm install --ignore-scripts
> ./scripts/install-fork.sh
Installed alias: ~/.local/bin/dusentrieb

Then run dusentrieb.
