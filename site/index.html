Inject into current chat? Yes / No

Agents love to interrupt you with questions. Duesentrieb doesn't. While you work, a second model watches the conversation in the background and collects the questions that are worth asking but not worth stopping for.

It deliberately hunts for the meta questions — "how should features like this generally be implemented?", "which abstraction do you actually prefer?" — instead of nitpicks about the diff currently in flight.

Run /open-questions whenever you feel like it. Each question comes with up to four multiple-choice answers, plus an escape hatch to write your own or discard it outright. And nothing touches your session behind your back: before an answer lands in the conversation, it asks whether to inject it into the current chat.

Questions are deduplicated, capped, and collected without ever blocking a turn. Your flow stays yours.

Go get it

Requires Node.js ≥ 22.19 and git:

one-liner
> curl -fsSL https://binarybaron.github.io/duesentrieb/install.sh | bash

Or do it by hand:

install
> git clone https://github.com/binarybaron/duesentrieb
> cd duesentrieb
> npm install --ignore-scripts
> ./scripts/install-fork.sh
Installed alias: ~/.local/bin/dusentrieb

Then run dusentrieb.
