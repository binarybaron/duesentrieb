Go get it

Requires Node.js ≥ 22.19 and git:

one-liner
> curl -fsSL https://binarybaron.github.io/duesentrieb/install.sh | bash

Or do it by hand:

install
> git clone https://github.com/binarybaron/duesentrieb
> cd duesentrieb
> npm install --ignore-scripts
> ./scripts/install-fork.sh
Installed alias: ~/.local/bin/dusentrieb

Then run dusentrieb.
